Calculadora Simples em Python

Este é um projeto que desenvolvi utilizando Python. A calculadora simples permite realizar operações de soma, subtração, multiplicação e divisão entre dois números fornecidos pelo usuário. O código foi desenvolvido para ser interativo, permitindo que o usuário escolha a operação desejada através de um menu simples no terminal.

Descrição do Código:

Este código em Python possui funções para cada operação matemática (soma, subtração, multiplicação e divisão), além de um menu interativo que recebe a escolha do usuário e executa a operação correspondente. O código lida com a entrada do usuário, verificando se a operação escolhida é válida e se os números inseridos são adequados.

Código Python:

def somar(x, y):
    return x + y

def subtrair(x, y):
    return x - y

def multiplicar(x, y):
    return x * y

def dividir(x, y):
    if y != 0:
        return x / y
    else:
        return "Erro! Divisão por zero."

# Função principal que exibe o menu e executa as operações
def calculadora():
    print("Selecione a operação desejada:")
    print("1. Soma")
    print("2. Subtração")
    print("3. Multiplicação")
    print("4. Divisão")

    # Recebe a escolha do usuário
    escolha = input("Digite o número da operação (1/2/3/4): ")

    # Verifica se a escolha é válida
    if escolha in ['1', '2', '3', '4']:
        num1 = float(input("Digite o primeiro número: "))
        num2 = float(input("Digite o segundo número: "))

        if escolha == '1':
            print(f"{num1} + {num2} = {somar(num1, num2)}")
        elif escolha == '2':
            print(f"{num1} - {num2} = {subtrair(num1, num2)}")
        elif escolha == '3':
            print(f"{num1} * {num2} = {multiplicar(num1, num2)}")
        elif escolha == '4':
            print(f"{num1} / {num2} = {dividir(num1, num2)}")
    else:
        print("Escolha inválida!")

# Executa a calculadora
calculadora()
